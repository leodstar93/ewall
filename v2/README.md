# EWALL V2 Lab

This folder contains the isolated visual redesign workspace for the app.

What lives here:

- standalone visual system for the experiment
- preview pages for the new shell and modules
- content and styling that should not affect the current experience

Routes:

- `/v2`
- `/v2/modules/[slug]`

Current app routes remain unchanged. When the redesign is mature enough, the
production shell can be migrated toward this structure.
